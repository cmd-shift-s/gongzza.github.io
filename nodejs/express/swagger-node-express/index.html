Express에 Swagger 붙이기

2017-07-18
2021-02-21
node.js, swagger-jsdoc, swagger-ui-express

Express에 Swagger를 붙여보려고 합니다.

Install

언제나 시작은 설치부터죠

  • swagger-ui-express: express에 swagger ui를 쉽게 붙일 수 있습니다.
  • swagger-jsdoc: Comment 형식으로 문서를 작성할 수 있습니다.

Swagger Specification

swagger-jsdoc를 이용해서 Swagger 명세서를 생성합니다.

apis 배열에 들어간 파일들의 주석을 분석합니다.

swaggerDefinition는 json이기 때문에 파일로 따로 분리할 수도 있고, production일 경우 basePath를 변경할 수 있습니다. 상세 정보는 여기를 참고하시면 됩니다.

host는 변경하지 않으면 현재 Domain을 따라서 갑니다.

Security Definitions

JWT 인증을 사용할 경우 다음과 같이 swaggerDefinition을 수정합니다.

security를 API마다 따로 적용할 수도 있지만, 여기서는 로그인 기반 서비스이기 때문에 전역으로 설정하였습니다.
자세한 사항은 여기 를 참고하시기 바랍니다.
